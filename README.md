# Run playwright

### run all scnearios
````
    npx playwright test
````

### run a specific scnario
````
    npx playwright test login.spec.ts
````

### run using playwright UI
````
    npx playwright test --ui 
````

# Create your .env file

```
    npm install dotenv --save  
```

# Using nvm

### changing the node version
````
    nvm use 18
````
